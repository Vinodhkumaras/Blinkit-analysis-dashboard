# Blinkit-analysis-dashboard
📊  Blinkit sales Analysis Dashboard  🍁 

 I have built an interactive sales analysis dashboard by using backend with Microsoft excel to analyze blinkit sales performance across different categories, outlets and locations..

✨  Key highlights :- 
1. Tracked key KPI's such as :-
    * Total sales
    * Number of orders
    * Average sales
    * Average rating

2. Analyzed sales performance based on
     * Item type (Category-wise analysis)
     * Outlet location
     * Outlet Size.

3. Discovered top-perforaming product categories contributing to overall sales.

4. Observed highest - performing Outlet locations across different tiers.

5. Evaluates Outlet types to determine which contributes the most to total sales.

6. Implemented slicers for dynamic filtering by :-
     * Outlet Size
     * Outlet location
     * Item type

This project helped me understand how data is used for real decision- making.....

 🛠 Tools used :- Microsoft excel in data as backend and with PowerBI with clean and maintained Dashboard with using( Pivot table, charts, slicers)
